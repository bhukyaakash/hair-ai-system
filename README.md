# hair-ai-system

AI-Powered Hairstyle Recommendation & Hair Health Assessment System.

## Stack
- Backend: FastAPI + PostgreSQL + TensorFlow/Keras
- Frontend: HTML/CSS/JavaScript (React-ready)
- Deployment: Docker, GCP, Kubernetes

## Quick start
```bash
./setup.sh
docker compose up --build
```

## Main directories
- `backend/`
- `frontend/`
- `deployment/`
- `docs/`
- `tests/`
